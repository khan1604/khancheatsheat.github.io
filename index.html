HTML's table reference

Table's tags

Tag Name Description
<table> table The wrapper element for all HTML tables.
<thead> table's heading defines column's heading in row.
<tbody> table's body All elements that appear in table, mostly used for styling.
<tr> table row defines each row (line)
<td> table data elements are used to define regular table cells
<tfoot> table footer The set of rows defining the footer in a table.

Table's Attributes

Atribbute Name Description
Colspan column span creating more complex table layouts or for merging cells
Rowspan row span Defines how many rows a td element should merge
